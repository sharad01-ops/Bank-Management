# Bank Management System

## Overview
The **Bank Management System** is a Python-based application designed to manage basic banking operations such as creating accounts, managing transactions, and viewing customer details. It provides a simple and efficient interface to simulate banking functionalities for both customers and administrators.

This project demonstrates core programming concepts including file handling, data persistence, modular coding, and user input validation. The backend is built in Python, and the interface can be extended using Streamlit for a user-friendly web-based dashboard.

---

## Features
- Create and manage customer accounts  
- Deposit and withdraw funds  
- Check balance and transaction history  
- Secure account verification  
- Streamlit-based interface for interactive user experience  

---

## Tech Stack
- **Language:** Python  
- **Framework:** Streamlit (for UI)  
- **Database:** File-based storage (can be extended to SQLite or MySQL)  
- **Version Control:** Git and GitHub
